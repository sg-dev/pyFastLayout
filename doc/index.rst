Welcome to Parallel FR's documentation!
=======================================

Introduction
------------

This extension implements the Fruchterman Reingold layout algorithm in a fast, parallel way. This extension is compatible with `igraph <http://igraph.org/>`_ as well as `graph-tool <https://graph-tool.skewed.de/>`_ for a demonstration of these use cases see :ref:`usage`.
